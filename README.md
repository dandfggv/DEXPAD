# DEXPAD — The Next‑Generation Note Editor for Windows

DEXPAD is a fast, modern, and secure note‑editing experience built for creators who want more than a basic notepad. Designed around a custom `.DEXPAD` file format, a smooth draggable UI, and seamless system‑tray integration, DEXPAD delivers a premium writing workflow with zero clutter and maximum control.

Whether you're drafting ideas, storing private notes, or building your own ecosystem around the DEXPAD format, this editor gives you the speed of Notepad with the polish of a dedicated writing tool.

---

## ✨ Features

- **Custom .DEXPAD File Format**  
  Purpose‑built format for clean, structured note storage.

- **Modern, Smooth UI**  
  Minimal, draggable window with a focus on comfort and speed.

- **System Tray Integration**  
  Keep DEXPAD running in the background and reopen it instantly from the tray.

- **Strict “No‑Use” License**  
  All rights reserved. No one may use, modify, or redistribute without your permission.

- **File Association Support**  
  Double‑click `.DEXPAD` files to open them directly in DEXPAD.

- **Full Windows Installer**  
  Inno Setup script with shortcuts, file associations, optional registry tweaks, and post‑install launch.

- **Lightweight & Fast**  
  Native Win32 C code with minimal overhead.
